# 🗡️ Rpg - Text RPG Game

A simple text-based RPG game built with Python, currently under development.

## 🎮 How to play

```bash
python jogo.py
```

## ⚔️ Features

- Choose your class at the start of the game
- Turn-based combat system
- In-game shop to buy items

## 🚧 Work in progress

This project is still under development. Planned features include:

- New enemies and boss fights
- New dungeons to explore
- More items and equipment
- Skills and abilities system

## 🛠️ Built with

- Python 3
